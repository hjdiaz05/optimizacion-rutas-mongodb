Descripción del Proyecto

Este proyecto permite gestionar información logística relacionada con rutas, clientes, conductores y vehículos.

La solución utiliza MongoDB Atlas como base de datos NoSQL para almacenar documentos de forma flexible y escalable.

El objetivo principal es optimizar las rutas de reparto, reducir tiempos de entrega y disminuir costos operativos.
